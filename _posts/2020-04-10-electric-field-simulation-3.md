在之前的文章中，模拟了两个点电荷 和 三个点电荷的电场、电势分布，我们来看下具体的例子。

如图，在 M、N 处固定两个等量同种电荷，两电荷均带正电。O 点是 MN 连线的中点，直线 PQ 是 MN 的中垂线。现有一带正电的试探电荷 q 自 O 点以大小是 $v_0$ 的初速度沿直线向 Q 点运动。若试探电荷 q 只受 M、N 处两电荷的电场力作用，则下列说法正确的是

<img src='/assets/images/2020-04-10-electric-field-simulation-3/fig.png' style='zoom:40%; margin: 0 auto; display: block;'/>
**A**. q 将做匀速直线运动  
**B**. q 的加速度将逐渐减小  
**C**. q 的动能将逐渐减小  
**D**. q 的电势能将逐渐减小  

这里，由于点电荷受到的力与距离平方成反比，q 并未处于不受力或受到平衡力，A 错。在 O 点受到的合力为 0，在无穷远处受到的合力为 0，因此，从 O 点移动至无穷远处过程中，合力会先变大后变小，无法判断加速度， B 错。从 O 点想无穷远处，速度在不断增加，动能也在增加， C 错。由于电场力做正功，电势能逐渐降低，D 对。

> 如果知道两个正电荷的电场强度分布，沿着电场线方向，电势降低， D 选项就容易判断了。

## 扩展

我们来计算下在中轴线上电场强度大小随着距离的变化。为方便直观展示，令 $k = 1$，两个点电荷分别位于 (0, 1)、(0, -1) 位置，带电量为 1。

<img src='/assets/images/2020-04-10-electric-field-simulation-3/Ex.png' style='zoom:40%; margin: 0 auto; display: block;'/>

随着距离的增加，电场强度大小先增加后减小，始终为正，即正试探电荷受力始终大于 0， 速度会不断增加。

OK，我们知道了两个点电荷的电场强度，也就知道了正试探电荷受到的加速度变化关系，那么我们能不能看看正试探电荷速度是怎么变化的呢？

## 变题

根据上面，来看个变题：

(多选) 如图，在 M、N 处固定两个等量同种电荷，两电荷均带正电。一光滑细杆在两个点电荷的中垂线上，光滑细杆上套有一个质量为 m = 10g 的带正电小球，小球所带电荷量 $q = 5.0 \cdot 10^{-4} C $。小球从 C 点由静止释放，其沿细杆由 C 经 B 向 A 运动的 $v-t$ 图如下所示。小球运动到 B 点时，速度图像的切线斜率最大，则下列说法正确的是

**A**. 在 O 点右侧杆上， B 点场强最大，场强大小为 $E = 12 V/m$  
**B**. 由 C 到 A 的过程中，小球的电势能先减小后变大  
**C**. 由 C 到 A 电势逐渐降低  
**D**. C、B 间的电势差 $U_{CB} = 0.9 V$  

<img src='/assets/images/2020-04-10-electric-field-simulation-3/fig2.png' style='zoom:40%; margin: 0 auto; display: block;'/>

在 B 点斜率最大，因此场强最大，场强大小可以用过 $Eq = ma$ 计算得出， A 正确。由 C 至 B 到 A 过程中，电场力做正功，电势能降低，B 错 C 对。BC 间的电势差可以使用功能原理： $qU = \frac{1}{2} mv^2 - 0$，D 正确。因此答案为 ACD。
