我们来探究下金属杆在磁场中的运动情况，借助程序，直观展示运动情况。来看以下的题：

(多选) 如图，两根电阻不计的平行金属导轨水平放置，一端与电阻相连。导轨 $x$ > 0 一侧存在沿 $x$ 方向均匀增大的稳恒磁场，其方向与导轨平面垂直，变化率 $k$ = 0.5 T/m，$x$ = 0 处磁场的磁感应强度 $B_0$ = 0.5 T。一根金属棒置于导轨上，并与导轨垂直。棒在外力作用下从 $x$ = 0 处以初速度 $v_0$ = 4m/s 沿导轨向右运动，运动过程中电阻消耗的功率不变。则 (    )

<img src='/assets/images/2020-04-11-rod-motion-in-magetic-field/fig.png' style='zoom:30%; margin: 0 auto; display: block;'/>

**A**. 金属棒在运动过程中产生的电动势不变  
**B**. 金属棒在运动过程中产生的安培力大小不变  
**C**. 金属棒在 $x$ = 3 m 处的速度为 2 m/s  
**D**. 金属棒在 $x$ = 3 m 处的速度为 1 m/s  

**解析** 这题由于磁场是随着 $x$ 增大而增大的，因此无法直观判断物理的运动，但题目中“运动过程中电阻消耗的功率不变” 说明了在运动过程中，电流是保持不变的，因此金属棒受到的安培力 $F = BIL$ 是逐渐增大的，A 对 B 错。而在 $x$ = 3 m 处的磁感应强度 B
 = 2 T，抓住电流保持不变这一条件，即电动势保持不变，$B_0Lv_0 = BLv$，可以计算出速度为 1 m/s。 D 正确。答案为 AD。

## 变题

我们来看一下变题。

如图，两根电阻不计的平行金属导轨水平放置，一端与电阻相连。导轨 $x$ > 0 一侧存在稳恒磁场，其方向与导轨平面垂直。一根金属棒置于导轨上，并与导轨垂直。棒在外力作用下从 $x$ = 0 处以初速度 $v_0$ = 4m/s 沿导轨向右运动，则金属棒的速度-时间图像可能是 (    )

<img src='/assets/images/2020-04-11-rod-motion-in-magetic-field/revise-1.png' style='zoom:40%; margin: 0 auto; display: block;'/>

**解析** 写出金属棒受到的安培力：

$$
F = BIL = \frac{B^2L^2v}{R} = ma = m\frac{dv}{dt}
$$

可以解出速度与时间的关系：

$$
v = v_0 e^{-\frac{B^2L^2}{mR}t}
$$

C 选项正确。

当然，可以从另一个角度思考，物体受到的安培力与运动方向相反，速度在不断减小，同时速度不断减小，也影响了切割磁感线的快慢，因此产生的感应电动势也在不断减小，即电流不断减小，金属棒安培力也不断减小，导致加速度不断减小。在 $v-t$ 图上查看曲线斜率，排查 A、B。 C 和 D 的区别在于速度会不会到达 0， 由于加速度在不断减小，速度会趋近 0。 C 选项正确。
